# Password Generator

## Description
Web-application which creates a randomly generated password based on answer to questions to the user:
* How many characters?
* Is a type of character Required, Forbidden, or Optional, for the following character types:
  * Upper-case alphabet
  * Lower-case alphabet
  * Numbers
  * Special Characters
